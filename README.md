so to compile and run this esp, u need to do the following things:
1. click on 'code' and download as a zip.
2. extract
3. download latest release of imgui and add it to the zip you just extracted
4. open developer command prompt for VS (you may not have this on your pc so just download it from the internet)
5. cd into the directory of extracted zip file (eg cd C:\Users\YOURUSER\Desktop\proecjt)
6. run build.bat from command prompt
7. it will spit out a file called r6 external esp or something along the lines
----- you have now compiled the .exe, we're not done yet.
you need to run the driver i have included, so go into 'driver' folder and run start_driver.bat as admin
after that, launch the game, go into borderless (otherwise it won't work)
when your game is fully loaded , and you're in the main menu , open the exe file you compiled earlier
if you did everything correctly, you should see that its connected to the driver and that its attached to the r6 process
now that its loaded, you can go use it in game. test it out in a custom game with bots or clear house first
